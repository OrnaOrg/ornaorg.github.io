What is Orna?

Orna is the good tool, designed for atomize Web UI.

Atomic CSS is a model or style of coding where each of properties has own style classes.

Atomic CSS is good for fast creating the readable and easy changeable User Interface. Also perfect for SPA(single page application).


What Orna do?

If you plug orna.js into html document you will have a new skill that give you write css properties with values in class attribute and it will means class and property in one and we will call it atom. You can write properties with few value and that calls molecules. Molecules make your work easy when properties have many value.


How Orna work?

Just write css properties into the class attribute then write underscore and value. Orna scan the html document, find atoms and define it directly to the element. Event in orna create with jQuery. For event you can use onclick, onmouseover, onfocus, onblur and other. Or simply add tail to the class name like: mouseover, mouseout, focus, blur, click. (width_100px_click, property_value_event)

Special classes

  1. Arial, arial - font


  2. Times, TimesNewRoman - font


  3. center - block elements in center by x axis


  4. textincenter - text and inline elements in center by x axis


  5. block - block element


  6. inline - inline element


  7. inlineblock - inline-block element


  8. uppercase - text in uppercase


  9. lowercase - text in lowercase


  10. capitalize - first symbol in uppercase


  11. hideatom - use for hide element from Orna, must be first in class attribute


  12. flexcenter-, flexstart-, flexend-, flexcenter|, flexstart|, flexend| - use for flexbox


  13. ornahelp - view the orna help info









OrnaJS functions for Atomize
